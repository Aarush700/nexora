# Nexora

AI-powered solutions for Aussie small businesses. Built with HTML, CSS & JS — hosted on GitHub Pages.

## What's Inside

- **Landing page** with hero, pricing cards & contact section
- **Enquiry form** — submissions go straight to email via [FormSubmit.co](https://formsubmit.co)
- **Nexora font** (custom brand typography)

## Quick Start

Open `index.html` in your browser — or run locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`
